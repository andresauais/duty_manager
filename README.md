# Duty Manager App

A full-stack task management application built with **React**, **TypeScript**, **Ant Design**, **Node**, **Express** and **PostgreSQL**. It allows users to add, edit, complete, and delete "duties" via a clean and responsive interface.

## 🛠 Tech Stack

### Frontend
- React (with Vite)
- TypeScript
- Ant Design
- React Testing Library + Vitest

### Backend
- Node.js
- Express.js
- PostgreSQL
- TypeScript
- Jest

## Setup

### Backend
```bash
cd backend
npm install
npm run start
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## 🧪 Run Frontend Tests
```bash
npm run test
```

## 🐳 Run with Docker
```bash
docker compose up --build
```

## 🚀 Features
- ✅ Add new duties
- 📝 Edit duties
- ❌ Delete duties
- ✔️ Mark duties as complete
